### 2. 系统架构

```
前端 (React + Vite + Tailwind)     后端 (FastAPI + LangGraph)
     :5173  ← 代理 →  :8000
         /chat/stream (SSE)       consultation_graph.py
         /sessions/{id}           9步状态机
         /upload/{id}             InMemorySaver
```

**数据流：**
1. 用户输入 → POST `/chat/stream` → LangGraph 状态机
2. 每步 AI 响应流式推送 → 前端逐字渲染 + 闪烁光标
3. 步骤完成 → SSE `done` 事件含 `current_step` → 前端更新进度条
4. 侧边栏从 `/sessions/{id}` 获取完整案件状态
